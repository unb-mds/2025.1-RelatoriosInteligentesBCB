Sobre o Projeto

Este projeto acadêmico tem como objetivo desenvolver uma solução para automatizar a geração de relatórios inteligentes a partir dos dados financeiros do IPEA/Banco Central do Brasil. Utilizando técnicas de inteligência artificial, a aplicação permite analisar grandes volumes de dados, identificar padrões e gerar relatórios detalhados e personalizados de forma eficiente. O projeto busca contribuir para a melhoria da transparência e acessibilidade das informações financeiras do IPEA/Banco Central do Brasil, facilitando a tomada de decisões e a compreensão do cenário econômico.
